# Agent Commerce Flow

Discovery is public; qualification and delivery are gated.

1. Seller or buyer agent reads the public trust surface and examples.
2. Aurora may provide a free no-secret snapshot: public evidence links, readiness status, three blind spots, and one scope question.
3. If useful, the prospect opens the seven-field Audit Request issue with public or redacted information.
4. Seller validates the seven required fields and accepts one fixed scope.
5. Seller issues a quote/payment mandate for `$99 Agent Fiscal Autonomy Audit`.
6. Buyer pays exact Base USDC and provides transaction hash.
7. Seller verifies chain, token, recipient, amount, tx success, quote id, expiry, and duplicate status without wallet custody.
8. Seller delivers the Markdown report and JSON summary.
9. Repeat audits are proposed only for event triggers, not generic reminders.

No payment route opens before qualification and explicit scope acceptance. The paid report is never public in this repo.
