# Capability Contract

Product: Agent Fiscal Autonomy Pack for OpenClaw.

Public plugin role: free proof, quote, entitlement, and install handoff wrapper.

Paid product role: derived-only strategy research database, retrieval chunks, strategy cards, proof reports, and safety/audit contracts.

The plugin may expose:

- public proof examples;
- schemas;
- quote and invoice templates;
- safety boundaries;
- install handoff instructions;
- public GitHub Pages documentation.

The plugin must not expose:

- paid database files;
- paid release package;
- buyer entitlements;
- private delivery links;
- raw vault content;
- wallet secrets;
- live trading instructions.
