# Safety Boundary

This plugin and the paid pack are research infrastructure. They are not a signal bot and not an exchange execution system.

Allowed:

- source-grounded retrieval;
- strategy-card formalization;
- risk-rule checks;
- paper-trading plans;
- operating-cost scenario analysis;
- proof-of-usefulness reports.

Blocked:

- guaranteed-profit claims;
- financial advice;
- exchange order placement;
- autonomous live trading;
- wallet secret handling;
- exposing raw private course materials;
- exposing private delivery links before entitlement.

Machine action stays blocked unless source ids, entry trigger, invalidation, and risk rule are present.
